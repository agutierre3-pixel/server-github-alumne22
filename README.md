Este proyecto consiste en la creación y configuración de un servidor Linux en la nube utilizando GitHub Codespaces.

El servidor ofrece:

Una web pública accesible sin autenticación.

Una web privada protegida con autenticación básica (usuario y contraseña).

Acceso remoto mediante SSH.

Gestión del proyecto y control de versiones con GitHub.
